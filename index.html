Sudoku Board

How to play

A sudoku puzzle consists of 3x3 groups, each containing 3x3 cells. Each cell must have a value between 1 - 9. Some cells are pre-filled, but you have to fill the rest following the rules below:

  • Each value (1 - 9) may appear only once in a row.
  • Each value (1 - 9) may appear only once in a column.
  • Each value (1 - 9) may appear only once in a group.

Once you fill all the cells following the above rules, the sudoku puzzle is complete.

New Puzzle

You can start a new puzzle with the following difficulties:

  • Easy
  • Normal
  • Hard
  • Empty

The Empty difficulty contains an empty puzzle. It's useful if you just want to enter a puzzle you found somewhere and solve it.

Finally, you can also select to automatically solve the puzzle.

Multiple colors

You can choose between 5 different colors while filling the cells. This is useful if you want to try a value and undo it leads to a dead end.

Undo

You can undo your last action or all the actions you did with the current color. This is especially useful if you tried a series of actions which led to a dead end.
